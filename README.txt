Este é um programa simples em Python feito como trabalho de faculdade (1º período de Sistemas de Informação). A ideia foi criar um sistema interativo no terminal que ajuda o usuário a acompanhar algumas informações básicas de saúde no dia a dia.

O programa funciona com um menu, onde o usuário escolhe o que quer calcular.

O que o programa faz
1. Cálculo de IMC

Calcula o IMC usando peso e altura informados pelo usuário.

2. Meta de hidratação

Calcula quanto de água a pessoa deve beber por dia com base no peso.

3. Gasto calórico

Faz uma estimativa de quantas calorias a pessoa gasta por dia, considerando:

peso
altura
idade
sexo
nível de atividade física

4. Meta de exercício diário

Calcula quanto tempo de exercício a pessoa deve fazer por dia, dependendo do objetivo:

manter-se ativo
emagrecer
musculação

5. Resumo diário

Mostra um resumo com os dados calculados anteriormente (IMC, hidratação e gasto calórico), caso o usuário tenha preenchido essas informações.

Como usar
Execute o código em Python
Escolha uma opção do menu (1 a 6)
Digite os dados pedidos
Veja o resultado no terminal
Observações
O programa usa fórmulas conhecidas (como IMC e TMB), mas os resultados são apenas estimativas
Os dados não são salvos, então tudo é perdido ao fechar o programa
Foi feito com foco em prática de lógica de programação
